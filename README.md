# Config Backup

A PowerShell script to backup \& restore your application configurations.



## Authors

* [@remusrigo](https://github.com/RemusRigo)
* [@remusrigo](https://sourceforge.net/u/remusrigo/profile)



## Installation

Just right click on "Config Backup.ps1 and select "Run with PowerShell" or run the "Config Backup.bat"



## Back-up \& Restore:

* Adobe CameraRAW
* Adobe Lightroom Classic
* Adobe Photoshop
* GeoSetter
* Lazarus IDE
* Microsoft PowerShell
* Microsoft Windows Terminal
* VLC
